# 🤖 AI Chat App

A full-stack chat application built using React, Tailwind CSS, and Node.js.

## 🚀 Features
- Real-time chat UI
- Mock AI responses
- Responsive design
- Clean UI with Tailwind

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- Node.js + Express

## 📦 Installation

```bash
npm install
npm run dev
